Installation du SRD
======================

On peut installer facilement le SRD en allant chercher d'abord sur GitHub::

  git clone https://github.com/creei-models/srd srd --recursive


On complète l'installation en allant dans srd/ et en lançant au terminal::

  python setup.py install

Dans un notebook ou un projet, on invoque SRD en ajoutant la commande suivante:

.. code:: ipython3

    import srd

Vous ne devriez pas travailler dans ce répertoire d'installation.

Pour désinstaller le module SRD, il suffit de rentrer la commande suivante dans le terminal::

  pip uninstall srd

Il faudra ensuite encore effacer le répertoire SRD local si l'on veut s'en débarrasser.
