Installation du SRD
======================

On peut installer facilement le SRD en utilisant pip::

  pip install srd

Le SRD dépend d'un autre package produit par la chaire, le SRPP, qui permet de simuler les revenus de pensions publiques (RPC et RRQ). Celui ci est installé automatiquement en installant le SRD.

Dans un notebook ou un projet, on invoque SRD en ajoutant la commande suivante:

.. code:: ipython3

    import srd

Pour désinstaller SRD, il suffit d'utiliser pip::

  pip uninstall srd

Si ce n'est pas possible:

1. Allez sur pypi.org, téléchargez les fichiers sources (extension .tar.gz) de srd et srpp
2. Extrayez les fichiers .tar.gz; sur windows, utilisez winzip ou, dans une fenêtre de commande, entrez (en remplaçant les adresses et noms de fichiers)::

  tar -xvzf C:\\PATH_TO_FILE\\FILE_NAME.tar.gz -C C:\\PATH_TO_FOLDER_EXTRACTION

3. Dans les dossiers qu’on vient d’extraire, *srd-x.x.x* et *srpp-x.x.x* (*x.x.x* étant le numéro de version), on trouve les répertoires *srd* et *srpp*
4. Placez *srd* et *srpp* dans le répertoire de votre choix

Pour invoquer SRD, il faut ajouter son adresse:

.. code:: ipython3

    import sys
    sys.path.append("C:\PATH\TO\FOLDER CONTAINING SRD")

    import srd

Si l'on ne veut plus SRD, il suffit d'effacer les répertoires *srd* et *srpp*.
