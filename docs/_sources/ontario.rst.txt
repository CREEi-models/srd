
Impôt de l'Ontario
------------------

Survol
^^^^^^

L'impôt de l'Ontario est pris en charge par le module *ontario*. Ce module contient un gabarit que nous documentons ci-dessous ainsi que des classes spécifiques chaque année.

.. currentmodule:: srd.ontario

La fonction *form* permet de choisir l'année du rapport d'impôt et ira tirer une instance du rapport pour cette année. L'instance est retournée par la fonction.

.. autofunction:: form

Gabarit du rapport
^^^^^^^^^^^^^^^^^^^^

Nous utilisons un gabarit afin de créer les rapports chaque année. Quand le programme change seulement dans ces paramètres, le rapport ira seulement chercher les nouveaux paramètres. Quand des fonctions changent, l'utilisateur n'a qu'à modifier les fonctions touchées (ou en rajouter). Toutes modifications de fonction survenues après l'année 2016 sont indiquées ci-dessous.

Nous reproduisons ici la spécification du gabarit. Il est calqué sur le rapport de 2016.

.. autoclass:: template
    :members: file, copy_fed_return, calc_tax, calc_non_refundable_tax_credits,
              get_age_cred, get_spouse_cred, get_cpp_contrib_cred, 
              get_pension_cred, get_disabled_cred, get_med_exp_cred, 
              get_donations_cred, surtax, div_tax_credit, tax_reduction, 
              lift_credit, calc_refundable_tax_credits, ocb, ostc, 
              calc_contributions, health_contrib

Le gabarit collecte les résultats sur un rapport d'impôt qui sera rattaché à la personne sous la forme d'un dictionnaire python. Ceci permet de différencier les attributs d'une personne qui font partie de son profil et les variables générées par l'impôt. C'est la fonction *create_return()* qui fait ce travail.

.. autofunction:: create_return
