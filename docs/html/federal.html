Impôt Fédéral¶

Survol¶

L’impôt fédéral est pris en charge par le module federal. Ce module contient un gabarit qui nous documentons ici-bas ainsi que des classes spécifiques à chaque année.

La fonction form permet de choisir l’année du rapport d’impôt et ira tirer une instance du rapport pour cette année. L’instance est retournée par la fonction.

srd.federal.form(year)¶

Fonction qui permet de sélectionner le formulaire d’impôt fédéral par année.

Paramètres

year (int) – année (présentement entre 2016 et 2020)

Renvoie

Une instance du formulaire pour l’année sélectionnée.

Type renvoyé

class instance

Gabarit du rapport¶

Nous utilisons un gabarit afin de créer les rapports à chaque année. Quand le programme change seulement dans ces paramètres, le rapport ira seulement chercher les nouveaux paramètres. Quand des fonctions changent, l’utilisateur n’a qu’à modifier les fonctions touchées (ou en rajouter).

Nous reproduisons içi la spécification du gabarit. Il est calcqué sur le rapport de 2016.

Le gabarit collecte les résultats sur un rapport d’impôt qui sera rattaché à la personne sous la forme d’une dictionnaire python. Ceci permet de différencier les attributs d’une personne qui font parti de son profil et les variables générées par l’impôt. C’est la fonction create_return() qui fait ce travail.
